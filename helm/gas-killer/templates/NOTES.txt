== DEPLOYMENT STATUS ==

The Gas Killer AVS stack is being deployed with the following components:

- L1 (Anvil) - Local blockchain with forked L1 state
{{- if .Values.l2.enabled }}
- L2 (Anvil) - Local blockchain with forked L2 state
{{- end }}
- {{ .Values.global.nodeCount }} operator nodes - BLS signing and task execution
- Router - Request routing and aggregation

== JOBS ==

Setup Job - deploys EigenLayer contracts and creates {{ .Values.global.nodeCount }} test operator accounts:
  kubectl logs job/{{ include "gas-killer.setup.fullname" . }}

{{- if .Values.bridge.enabled }}
Bridge Job - bridges operator state from L1 to L2 (runs after setup):
  kubectl logs job/{{ include "gas-killer.bridge.fullname" . }}
{{- end }}

{{- if and .Values.l2.enabled .Values.yieldDistribution.enabled }}
Yield Distribution Job - configures and verifies YieldDistributor on L2 (runs after bridge):
  kubectl logs job/{{ include "gas-killer.yield-distribution.fullname" . }}
{{- end }}

Monitor all jobs:
  kubectl get jobs -l app.kubernetes.io/instance={{ .Release.Name }}

== ACCESSING SERVICES ==

L1 RPC:
  kubectl port-forward svc/{{ include "gas-killer.l1.fullname" . }} 8545:{{ .Values.l1.service.port }}

{{- if .Values.l2.enabled }}
L2 RPC:
  kubectl port-forward svc/{{ include "gas-killer.l2.fullname" . }} 8546:{{ .Values.l2.service.port }}

Router:
  kubectl port-forward svc/{{ include "gas-killer.router.fullname" . }} 3000:{{ .Values.router.service.port }}
